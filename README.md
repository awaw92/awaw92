## 📂 Projekty

### 1. Google Search Front-End (CS50 Web Project)

Projekt polegający na stworzeniu front-endu imitującego Google Search, Google Image Search oraz Google Advanced Search.

Aplikacja składa się z trzech stron:
- standardowe wyszukiwanie Google
- wyszukiwanie obrazów
- wyszukiwanie zaawansowane

Każda z nich wysyła zapytania do Google poprzez odpowiednio skonfigurowane formularze HTML (GET parameters).

Użytkownik może:
- wpisać zapytanie i przejść do wyników wyszukiwania Google
- przejść do wyszukiwania obrazów
- skorzystać z wyszukiwania zaawansowanego (np. dokładne frazy, wykluczanie słów)
- użyć przycisku „I’m Feeling Lucky” przekierowującego bezpośrednio do pierwszego wyniku

**Technologie:** HTML, CSS

### 2. Wiki Encyclopedia (CS50 Web Project)

Aplikacja internetowa typu wiki stworzona w frameworku Django, umożliwiająca przeglądanie, tworzenie oraz edycję haseł encyklopedycznych zapisywanych w formacie Markdown.

Każda strona encyklopedii jest dynamicznie generowana na podstawie plików Markdown i konwertowana do HTML podczas wyświetlania.

Funkcjonalności aplikacji:
- przeglądanie istniejących wpisów (/wiki/TITLE)
- wyszukiwanie wpisów (dokładne dopasowanie oraz wyszukiwanie fragmentów)
- tworzenie nowych stron encyklopedii
- edycja istniejących wpisów
- losowanie losowej strony
- obsługa błędów (strona 404 dla nieistniejących wpisów)

Dodatkowo:
- obsługa konwersji Markdown → HTML
- dynamiczne generowanie treści stron

**Technologie:** Python, Django, HTML, CSS, Markdown

**Demo:** [link do YouTube]

### 3. Auction Site (CS50 Web Project – Commerce)

Aplikacja e-commerce typu aukcyjnego stworzona w Django, inspirowana platformą eBay. Umożliwia użytkownikom tworzenie aukcji, składanie ofert, komentowanie oraz zarządzanie obserwowanymi ogłoszeniami.

Aplikacja opiera się na systemie użytkowników oraz relacjach między modelami (aukcje, oferty, komentarze).

Funkcjonalności:
- tworzenie nowych aukcji z tytułem, opisem, ceną początkową oraz opcjonalnym obrazem i kategorią
- przeglądanie aktywnych aukcji na stronie głównej
- szczegółowa strona aukcji z aktualną ceną i informacjami
- system licytacji (bids) z walidacją minimalnej wartości
- możliwość dodania/usunięcia aukcji z listy obserwowanych (watchlist)
- zamykanie aukcji przez właściciela i wyłonienie zwycięzcy
- system komentarzy pod aukcjami
- przeglądanie aukcji według kategorii
- integracja z panelem administracyjnym Django

**Technologie:** Python, Django, SQLite, HTML, CSS

### 4. Mail Client (CS50 Web Project)

Aplikacja typu single-page email client stworzona w JavaScript, wykorzystująca API do obsługi skrzynki mailowej.

Projekt symuluje działanie systemu poczty elektronicznej, umożliwiając wysyłanie, odbieranie oraz zarządzanie wiadomościami bez przeładowywania strony.

Funkcjonalności:
- wysyłanie wiadomości e-mail (Send Mail) poprzez API
- wyświetlanie skrzynek: Inbox, Sent, Archive
- dynamiczne ładowanie wiadomości (SPA – bez przeładowania strony)
- podgląd treści pojedynczej wiadomości
- oznaczanie wiadomości jako przeczytane / nieprzeczytane
- archiwizowanie i przywracanie wiadomości
- odpowiadanie na wiadomości (auto-uzupełnianie treści i tematu)

Aplikacja w pełni opiera się na JavaScript oraz komunikacji z backendowym API (fetch).

**Technologie:** JavaScript, HTML, CSS, REST API

### 5. World Geography Quiz (Final Project)

Aplikacja webowa typu quiz stworzona w Django, umożliwiająca sprawdzanie wiedzy z geografii świata poprzez interaktywny test z różnymi poziomami trudności.

Użytkownik wybiera poziom trudności, odpowiada na zestaw losowych pytań, a po zakończeniu otrzymuje wynik oraz miejsce w rankingu.

Funkcjonalności:
- quiz z losowo wybieranymi pytaniami (Easy / Medium / Hard)
- zarządzanie przebiegiem quizu przy użyciu Django sessions
- system punktacji i zapisywanie wyników w bazie danych
- ranking graczy (leaderboard) według poziomu trudności
- dynamiczna obsługa odpowiedzi (JavaScript – feedback, animacje)
- szczegółowe podsumowanie odpowiedzi po zakończeniu quizu

Aplikacja łączy backend Django (logika, modele, sesje) z dynamicznym frontendem opartym o JavaScript.

**Technologie:** Python, Django, JavaScript, HTML, CSS, SQLite
